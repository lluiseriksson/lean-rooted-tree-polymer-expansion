# Release notes: v1.0.0

This is the first publication release of the companion artifact for
**Marked Rooted-Tree Summation for Polymer Cluster Expansions with Holes: A
Lean 4 Formalization**.

## Included

- 16-page manuscript PDF and LaTeX source;
- optional graphical abstract;
- three stable Lean publication endpoints;
- exact upstream, Lean, Mathlib, and source-blob pins;
- theorem-to-paper map and source-provenance excerpts;
- GitHub Actions builds for Lean and LaTeX;
- axiom oracle, placeholder audit, PDF preflight, and artifact checks;
- release ZIP generation and SHA-256 manifests;
- cover letter, metadata, declarations, and publishing-agent handoff.

## Mathematical scope

The release formalizes finite rooted-tree and target-sensitive second-Ursell
bounds. It does not contain a model-specific proof of the raw Yang-Mills
activity estimate, `hRpoly`, a continuum construction, or a mass gap theorem.
