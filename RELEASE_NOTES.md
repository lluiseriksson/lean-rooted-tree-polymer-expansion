# Release notes: v2.0.0

Version 2.0.0 is the documentation-integrated publication release.

## Included

- complete scholarly article under `docs/paper/`;
- three stable Lean publication endpoints;
- exact upstream, Lean, and Mathlib pins;
- strict documentation build and GitHub Pages workflow;
- theorem map, source provenance, oracle, evaluator guide, and reviewer FAQ;
- deterministic source ZIP generation with SHA-256 manifests;
- governance, security, metadata, and migration files for public release.

## Deliberately removed

- standalone manuscript PDF and duplicated LaTeX source;
- graphical-abstract PDF duplicate;
- nested copies of prior ZIP/PDF release artifacts.

## Mathematical scope

The release formalizes finite rooted-tree and target-sensitive second-Ursell
bounds. It does not contain a model-specific proof of the raw Yang--Mills
activity estimate, `hRpoly`, a continuum construction, or a mass gap theorem.
