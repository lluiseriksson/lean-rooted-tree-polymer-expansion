# Repository identity history

The public repository is:

```text
lluiseriksson/lean-rooted-tree-polymer-expansion
```

The documentation is published at:

```text
https://lluiseriksson.github.io/lean-rooted-tree-polymer-expansion/
```

The repository was originally released as `marked-rooted-closure` and was
renamed after v2.0.0 so that the public URL explains the implementation
language, proof method, and mathematical setting. GitHub redirects ordinary
repository links from the former slug, but versioned releases and archived
citations remain the preferred permanent references.

The Lean package and namespace intentionally remain `MarkedRootedClosure` to
preserve downstream imports and theorem references.

The idempotent maintenance helper
[`scripts/rename_repository.py`](scripts/rename_repository.py) remains in the
repository for future owner or slug migrations. It defaults to a dry run and
must be followed by the complete static and documentation audits.
