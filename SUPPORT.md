# Support

Use GitHub issues for reproducibility failures, documentation defects, theorem
mapping questions, or narrowly scoped mathematical concerns. Include the exact
repository revision, command, environment, and smallest relevant log.

Do not disclose security or release-integrity vulnerabilities in a public
issue; follow [`SECURITY.md`](SECURITY.md).

General questions about the broader Yang--Mills programme belong in the
upstream repository when they are not specific to the three theorem endpoints
published here.
