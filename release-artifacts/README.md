# Release artifacts

This directory preserves the three externally supplied files used to assemble
this local repository:

- `marked-rooted-closure-v1.0.0.zip`: original source release bundle.
- `marked-rooted-closure-v1.0.0.zip.sha256`: SHA-256 sidecar for the ZIP.
- `marked-rooted-closure-paper.pdf`: externally supplied copy of the paper PDF.

The PDF copy is byte-identical to `paper/main.pdf`.
