# Roadmap

The v2.0.0 article and theorem interface are intended to remain stable.
Potential future work belongs in explicitly versioned releases.

1. Record a green clean-cache Lean CI run and archive DOI for v2.0.0.
2. Add a publisher-neutral print renderer generated from the same Markdown
   sources only if a venue requires one.
3. Formalize a sharper tree-profile constant if a genuine Catalan or restricted
   incompatibility enumeration becomes available.
4. Instantiate the weight split from primary Balaban/Dimock source statements.
5. Prove the concrete raw gauge activity estimate in the upstream Yang--Mills
   analysis programme.
6. Generalize the finite tree engine to other target-preserving polymer
   geometries.

No roadmap item is part of the theorem claimed in v2.0.0.
