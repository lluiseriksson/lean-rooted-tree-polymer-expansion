# Roadmap

The publication artifact is frozen at v1.0.0. Potential future work belongs in
separate releases or repositories.

1. Formalize a sharper tree-profile constant if an exact Catalan or restricted
   incompatibility enumeration is available.
2. Instantiate the weight split from primary Balaban/Dimock source statements.
3. Prove the concrete raw gauge activity estimate in a separate Yang-Mills
   analysis project.
4. Generalize the finite tree engine to other target-preserving polymer
   geometries.

No roadmap item is part of the theorem claimed in v1.0.0.
